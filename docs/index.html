Crypto Sentiment Bot

Combines real-time sentiment from major crypto news RSS feeds (60%) with the Fear & Greed Index (40%) to generate a definitive market signal.

How it works:
Headlines from CoinDesk, Cointelegraph, and Decrypt are fetched via RSS and scored locally using the AFINN lexicon (sentiment library). This produces a News Sentiment score (−1 to +1). Combined with the Fear & Greed Index from Alternative.me, the weighted aggregate (News RSS 60% + F&G 40%) is mapped to a discrete signal: BULLISH (≥ +0.25), BEARISH (≤ −0.25), or NEUTRAL. No external API keys are required for the sentiment analysis — it runs entirely client-side.
